# P109 請小聲喔

**環境音量偵測與智慧提醒系統**  
**P109 Please Keep It Quiet**  
**A Smart Sound-Level Detection and Alert System**

## 專案定位

P109「請小聲喔」是一套環境音量偵測與智慧提醒系統。它將環境聲量轉換成可觀察、可提醒、可記錄的管理資料，協助使用者理解公共空間的聲音狀態，進而改善生活品質、工作品質、學習專注與空間舒適度。

## V1.2 功能

- 即時偵測環境音量
- 顯示近似 dB 數值與 20 秒可調式聲量波形圖
- 可設定警戒門檻，波形超標段以紅色顯示
- 採用「時間窗內累積超標比例」作為預設提醒邏輯
- 可設定觀察時間窗，例如 15 秒
- 可設定超標比例，例如 30%
- 可設定警報冷卻時間
- 可設定顯示校正值
- 超標時播放提醒音並顯示警示畫面
- 記錄超標事件時間、最高聲量、觀察時間窗與超標比例
- 清楚揭露隱私設計

## 隱私設計

本系統 V1 僅偵測環境音量強度：

- 不錄音
- 不儲存音檔
- 不進行語音辨識
- 不辨識說話者身分
- 不記錄人聲內容

事件紀錄只保存時間、最高聲量、觀察時間窗與超標比例。

## 使用方式

1. 將本資料夾上傳到 GitHub Pages 或其他靜態網站空間。
2. 用瀏覽器開啟 `index.html`。
3. 按下「開始偵測」。
4. 允許瀏覽器使用麥克風。
5. 調整警戒門檻、觀察時間窗與超標比例。

## 注意事項

- 網頁版需要使用 HTTPS 或 localhost 才能正常取得麥克風權限。
- dB 顯示為近似值，不等同於校正過的專業聲級計。
- 不同手機、平板、筆電的麥克風靈敏度不同，正式使用前應搭配分貝計校正。
- 手機瀏覽器若鎖屏或切換到背景，偵測可能中斷。長時間監測建議未來開發 Android App 版本。

## V1.2 更新重點

原本的「連續超標 N 秒」改為更適合人聲與討論聲的「時間窗內累積超標比例」：

- 預設觀察時間窗：15 秒
- 預設超標比例：30%
- 意思是最近 15 秒內，如果約 30% 以上的偵測點超過門檻，才觸發提醒。

這樣可以避免單次咳嗽、關門聲或椅子聲造成誤報，也比嚴格連續超標更符合真實聊天與討論聲的波動特性。
