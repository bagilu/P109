# P109 請小聲喔

**環境音量偵測與智慧提醒系統**  
**P109 Please Keep It Quiet**  
**A Smart Sound-Level Detection and Alert System**

## 專案定位

P109「請小聲喔」是一套環境音量偵測與智慧提醒系統。它將環境聲量轉換成可觀察、可提醒、可記錄的管理資料，協助使用者理解公共空間的聲音狀態，進而改善生活品質、工作品質、學習專注與空間舒適度。

## V1.1 功能

- 即時偵測環境音量
- 顯示近似 dB 數值
- 以 20 秒預設時間窗顯示聲量波形圖
- 可調整波形時間窗：5、10、15……60 秒
- 在波形圖中顯示警戒門檻線
- 超過門檻的波形區段以紅色顯示
- 可設定警戒門檻
- 可設定連續超標秒數
- 可設定警報冷卻時間
- 可調整提醒音音量
- 可設定顯示校正值
- 超標時播放提醒音並顯示警示畫面
- 記錄超標事件時間、最高聲量與持續秒數
- 清楚揭露隱私設計

## V1.1 修正重點

1. 提醒音改為較明顯的三段式 beep，並使用獨立 Web Audio Context，以提高瀏覽器播放成功率。
2. 新增「提醒音量」調節鈕，可設為 0% 至 100%。
3. 將原本跳動過快的音量條改為即時聲量波形圖。
4. 波形圖預設呈現約 20 秒聲量資料，最新聲量由左側進入，往右側移動。
5. 波形圖顯示門檻線；超過門檻的波段顯示為紅色。

## 隱私設計

本系統 V1 僅偵測環境音量強度：

- 不錄音
- 不儲存音檔
- 不進行語音辨識
- 不辨識說話者身分
- 不記錄人聲內容

事件紀錄只保存時間、最高聲量與持續秒數。

## 使用方式

1. 將本資料夾上傳到 GitHub Pages 或其他靜態網站空間。
2. 用瀏覽器開啟 `index.html`。
3. 按下「開始偵測」。
4. 允許瀏覽器使用麥克風。
5. 調整警戒門檻、超標秒數、提醒音量與波形時間窗。
6. 可先按「測試提醒音」確認裝置音量與瀏覽器播放權限。

## 注意事項

- 網頁版需要使用 HTTPS 或 localhost 才能正常取得麥克風權限。
- dB 顯示為近似值，不等同於校正過的專業聲級計。
- 不同手機、平板、筆電的麥克風靈敏度不同，正式使用前應搭配分貝計校正。
- 若提醒音仍聽不到，請檢查系統音量、瀏覽器分頁是否被靜音、手機是否處於靜音模式或藍牙輸出是否連到其他裝置。
- 手機瀏覽器若鎖屏或切換到背景，偵測可能中斷。長時間監測建議未來開發 Android App 版本。
